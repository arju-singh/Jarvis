🧩 Where each fits in a project

Understand & reply to the user→ LLM / Brain API (Gemini, Groq, Ollama…)
Speak the reply out loud→ TTS API (Piper, ElevenLabs, Azure…)
Hear the user (speech → text)→ STT (Whisper — runs locally in Jarvis)
Fetch live facts (weather, news, flights)→ Data APIs (Open-Meteo, search, Amadeus…)
Answer over your own docs/data→ Embeddings + search (Cohere, OpenRouter…)

⚠️ The one rule that keeps it legal

  • Sign up with your own account, use your own key, stay within the free quota, and follow each provider's terms (usually one free account per person, no reselling, no abuse).
  • Never use keys you found posted online or that belong to someone else.
